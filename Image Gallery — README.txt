Project Overview

This project is a Responsive Image Gallery built using HTML, CSS, and JavaScript. 
It showcases a collection of images with smooth hover effects, navigation, and responsive design, making it suitable for different screen sizes.

Features

Responsive Layout: Adjusts automatically to different screen sizes (desktop, tablet, mobile).

Hover Effects: Cards lift slightly with shadow and overlay effect on hover.

Navigation / Lightbox: Click on an image to view in a lightbox with next/previous navigation and caption display.

Category Filters: Filter images by categories (Nature, City, People, All).

Image Style Toggle: Apply CSS filters such as grayscale, sepia, and blur.

Accessibility: Keyboard navigation support and focusable image cards.

Lazy Loading: Images load only when needed to optimize performance.

Notes

All images are sample images from Unsplash; replace with your own images if desired.

The gallery is fully functional without any external libraries.
