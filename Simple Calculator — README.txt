Simple Calculator — README
Project Overview

This project is a Simple Calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

Features

Basic Arithmetic Operations: Supports +, -, ×, ÷.

Clear and Reset: Option to clear the current input and start fresh.

Responsive Layout: Works on desktop and mobile devices.

Keyboard Input Support: Users can type numbers and operators using the keyboard.

Hover and Button Effects: Buttons have hover effects for better UX.

Notes

The calculator is fully functional without any external libraries.

All operations are performed on the client side.
